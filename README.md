# WeCare Beauty Store Inventory System

This is a Python-based inventory management system designed for a beauty store.  
It helps manage products, process sales, restock inventory, and generate invoices efficiently.

## Features
- Display available products
- Process product sales
- "Buy 3 Get 1 Free" promotional logic
- Restock existing and new products
- Generate and save invoices
- Automatically update inventory

## Technologies Used
- Python
- File Handling
- Modular Programming

## Project Structure
- main.py → Main program and menu system
- read.py → Loads product data from file
- operation.py → Handles sales and restocking logic
- write.py → Saves data and generates invoices
- wecare_products.txt → Stores product details

## How to Run
1. Clone or download this repository
2. Open the project folder
3. Run the main file:
   python main.py

4. Follow the menu instructions in the terminal
